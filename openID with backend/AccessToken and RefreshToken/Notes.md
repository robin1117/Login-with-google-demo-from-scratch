
(OpaqueToken)Access Token – A short-lived credential (usually expires in 1 hour) that proves you have permission to access specific Google APIs like googleDrive, Emails, Youtube etc on behalf of a user. It's sent with every API request. This is Bearer type of token. 

(OpaqueToken)Refresh Token – A long-lived credential (valid until revoked) used only to get a new Access Token when the old one expires, without requiring the user to log in again.
